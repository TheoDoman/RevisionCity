## ğŸ†˜ TROUBLESHOOTING

### **Common Issues**

**Issue:** Can't deploy to Vercel
- **Solution:** Check Vercel CLI is installed: `vercel --version`
- Install if needed: `npm install -g vercel`

**Issue:** Environment variables not working
- **Solution:**
  1. Check spelling in Vercel dashboard
  2. Ensure "Production" is selected
  3. Redeploy after adding variables

**Issue:** Payment not working
- **Solution:**
  1. Verify Stripe keys are correct
  2. Check webhook URL matches your domain
  3. Look for events in Stripe dashboard

**Issue:** AI test generator failing
- **Solution:**
  1. Check Anthropic API key is valid
  2. Verify API rate limits not exceeded
  3. Check error logs in Vercel

**Full troubleshooting:** See `LAUNCH-TODAY-PLAN.md` (bottom section)
